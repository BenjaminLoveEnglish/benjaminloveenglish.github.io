拒绝同事额外工作请求的表达方式

中文 中式表达 英式表达 美式表达
我现在很忙，没时间。 I am very busy now, no time. I'm rather tied up at the moment, I'm afraid. I'm completely swamped right now.
对不起，我这周工作量太大了。 Sorry, my workload is too big this week. I'm terribly sorry, but I've got rather a full schedule this week. Sorry, but I'm maxed out with assignments this week.
我能不能下周再帮你？ Can I help you next week instead? Might I be able to assist you next week instead? Would it work if I helped you out next week instead?
你能不能找别人帮忙？ Can you find someone else to help? I wonder if perhaps someone else might be available to lend a hand? Any chance you could find someone else to take this on?
这个任务很紧急吗？ Is this task very urgent? I'm just wondering about the urgency of this matter? How time-sensitive is this project?
我可以做一部分，但不能全部完成。 I can do some part, but cannot finish all. I could manage a portion of this, but completing the entire task wouldn't be feasible, I'm afraid. I can tackle part of this, but there's no way I can handle the whole thing.
如果你能帮我处理其他工作，我就能接受这个任务。 If you can help me deal with other work, I can accept this task. I could take this on if you'd be willing to assist me with some of my current responsibilities. I could make this work if you help me clear some other stuff off my plate.
我不擅长这个，可能做不好。 I am not good at this, maybe cannot do well. This isn't really my area of expertise, so I might not be the best person for the job. This isn't exactly in my wheelhouse, so I probably wouldn't do it justice.
我需要先问问经理我是否可以接受新任务。 I need to first ask manager if I can accept new task. I'd need to check with my manager whether I can take on any additional projects at present. I'd have to run this by my boss to see if I can add anything else to my workload right now.
我今天已经安排了很多工作，明天再说吧。 I have arranged many works today, let's talk tomorrow. I've quite a full day planned already. Perhaps we could discuss this tomorrow? My day is already packed. Can we circle back on this tomorrow?
